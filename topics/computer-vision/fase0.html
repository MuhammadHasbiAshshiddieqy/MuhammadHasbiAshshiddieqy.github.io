0.1 Pengenalan Computer Vision // dari piksel ke pemahaman

Computer Vision (CV) adalah bidang AI yang memungkinkan komputer untuk memahami konten visual — gambar, video, dan kamera secara real-time. Bukan sekadar menyimpan piksel, tapi menginterpretasikan: "ini anjing", "mobil di kiri berjalan 60 km/h", "sel tumor terdeteksi di koordinat ini".

CV adalah salah satu pencapaian terbesar AI modern. Dalam 10 tahun terakhir, akurasi pengenalan gambar melampaui kemampuan manusia di beberapa benchmark. Aplikasinya ada di mana-mana: kamera HP, mobil otonom, diagnosa medis, inspeksi pabrik, hingga AR/VR.

Mengapa CV sulit secara historis?

Manusia bisa mengenali wajah dari sudut berbeda, cahaya berbeda, dan sebagian tertutup — secara instan. Komputer harus belajar ini dari data. Tantangan utama: variasi intra-class (anjing bermacam bentuk), inter-class similarity (serigala vs husky), occlusion (objek tertutup), dan viewpoint variation. Pre-deep learning, pendekatan hand-crafted features (SIFT, HOG) efektif tapi terbatas.

Revolusi Deep Learning untuk CV

2012 adalah titik balik: AlexNet menang ImageNet dengan margin besar menggunakan CNN. Sejak itu akurasi terus meningkat pesat. 2020: Vision Transformer (ViT) menunjukkan bahwa arsitektur Transformer yang sukses di NLP juga bekerja untuk vision. 2021–2024: CLIP, DALL-E, Stable Diffusion, SAM — model generatif dan multimodal mendefinisikan ulang apa yang mungkin.

Task utama dalam CV

  • Classification — "gambar ini berisi apa?"
  • Detection — "di mana objek X berada?" (bounding box)
  • Segmentation — "piksel mana yang merupakan objek X?"
  • Pose Estimation — "di mana sendi-sendi tubuh?"
  • Generation — "buat gambar yang menggambarkan X"
  • Tracking — "ikuti objek ini lintas frame video"

Stack teknologi CV modern

  • PyTorch — framework utama riset & produksi
  • OpenCV — image processing, preprocessing, classical CV
  • torchvision — datasets, transforms, pretrained models
  • Hugging Face — pretrained vision models (ViT, CLIP, SAM)
  • ONNX / TensorRT — deployment & optimasi
  • Albumentations — augmentasi data cepat

Peta kurikulum ini

1
Image Processing Klasik
Fondasi: piksel, filter, konvolusi manual, edge detection, transformasi geometris. Tanpa ini, model deep learning adalah "black box".
2
Deep Learning untuk CV
CNN dari scratch, transfer learning, Vision Transformer, augmentasi data, loss functions, dan evaluasi model.
3
Detection & Segmentation
YOLO untuk real-time detection, semantic segmentation, Mask R-CNN untuk instance segmentation, dan pose estimation.
4A
Vision Systems (Jalur Produksi)
Optimasi model, quantization, ONNX, edge deployment di Jetson/RPi, real-time pipeline production-grade.
4B
Generative CV (Jalur Kreatif)
CLIP, diffusion models, Stable Diffusion fine-tuning, image editing, dan video generation.
5
Production & Deployment
End-to-end system design, serving dengan Triton, monitoring drift, anotasi & active learning.
