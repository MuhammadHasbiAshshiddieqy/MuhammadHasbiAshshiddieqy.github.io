1.1 Representasi Gambar & Piksel // gambar adalah array angka

Sebuah gambar digital adalah array 3 dimensi: (height, width, channels). Setiap piksel adalah satu atau beberapa angka — untuk grayscale satu nilai 0–255, untuk RGB tiga nilai. Memahami representasi ini adalah fondasi semua operasi CV.

Format gambar dalam NumPy

OpenCV membaca gambar sebagai array NumPy dengan format BGR (bukan RGB — urutan channel berbeda). Untuk konversi: cv2.cvtColor(img, cv2.COLOR_BGR2RGB). Nilai default: uint8 (0–255). Saat dikirim ke model neural network, biasanya dinormalisasi ke float32 (0.0–1.0).

Channels dan bit depth

Grayscale: shape (H, W) — 1 nilai per piksel. RGB/BGR: shape (H, W, 3). RGBA: shape (H, W, 4) dengan alpha channel. 16-bit: medical imaging sering pakai uint16 (0–65535) untuk detail lebih tinggi. Model biasanya bekerja di float32 setelah normalisasi.

Pythonimage_basics.py
import cv2
import numpy as np

# Baca gambar
img_bgr = cv2.imread("foto.jpg")     # shape: (H, W, 3), dtype: uint8
img_rgb = cv2.cvtColor(img_bgr, cv2.COLOR_BGR2RGB)
img_gray = cv2.cvtColor(img_bgr, cv2.COLOR_BGR2GRAY)  # shape: (H, W)

print(img_rgb.shape)    # (480, 640, 3)
print(img_gray.dtype)   # uint8
print(img_rgb.min(), img_rgb.max())  # 0 255

# Akses piksel individual
r, g, b = img_rgb[100, 200]   # piksel baris 100, kolom 200

# Crop region of interest
roi = img_rgb[50:150, 100:300]   # [y1:y2, x1:x2]

# Normalisasi untuk model neural network
img_float = img_rgb.astype(np.float32) / 255.0  # range [0, 1]

# Normalisasi ImageNet (mean/std per channel)
mean = np.array([0.485, 0.456, 0.406])
std  = np.array([0.229, 0.224, 0.225])
img_normalized = (img_float - mean) / std
1.2 Filter & Konvolusi Klasik // operasi matematis di balik CNN

Konvolusi adalah operasi inti yang sama dipakai di CNN dan image processing klasik. Sebuah kernel (matrix kecil) digeser di atas gambar, menghasilkan output berupa fitur tertentu. Gaussian blur, sharpening, emboss — semuanya adalah konvolusi dengan kernel berbeda. CNN hanya belajar kernel-nya secara otomatis dari data.

Cara kerja konvolusi 2D

Untuk setiap posisi kernel di atas gambar: lakukan element-wise multiplication antara kernel dan patch gambar di bawahnya, lalu jumlahkan hasilnya — satu angka per posisi. Geser kernel satu piksel (stride=1), ulangi. Output: feature map dengan ukuran ≈ input (tergantung padding).

Kernel umum dan efeknya

  • Gaussian blur (5×5) — menghaluskan noise, memburamkan detail
  • Laplacian — mendeteksi tepi (semua arah)
  • Sobel X/Y — gradien horizontal / vertikal
  • Sharpen — mempertajam tepi
  • Identity — output = input (tidak ubah apa pun)

Padding dan stride

Valid padding: tidak ada padding, output lebih kecil dari input. Same padding: padding dengan nol di tepi sehingga output sama ukurannya dengan input. Stride: seberapa jauh kernel bergerak per langkah — stride=2 menghasilkan output setengah ukuran (down-sampling).

Pythonconvolution_filters.py
import cv2
import numpy as np

img = cv2.imread("foto.jpg", cv2.IMREAD_GRAYSCALE)

# Gaussian blur — mengurangi noise
blurred = cv2.GaussianBlur(img, (5, 5), sigmaX=0)

# Sharpening kernel manual
kernel_sharp = np.array([[0,-1,0],
                            [-1,5,-1],
                            [0,-1,0]], dtype=np.float32)
sharpened = cv2.filter2D(img, -1, kernel_sharp)

# Median filter — noise removal tanpa blur tepi
median = cv2.medianBlur(img, ksize=5)

# Implementasi konvolusi manual dengan NumPy (untuk pemahaman)
def convolve2d_manual(img: np.ndarray, kernel: np.ndarray) -> np.ndarray:
    kh, kw = kernel.shape
    pad = kh // 2
    padded = np.pad(img, pad, mode='constant')
    out = np.zeros_like(img, dtype=np.float32)
    for i in range(img.shape[0]):
        for j in range(img.shape[1]):
            out[i, j] = (padded[i:i+kh, j:j+kw] * kernel).sum()
    return np.clip(out, 0, 255).astype(np.uint8)

1.3 Edge Detection // menemukan batas antar region




Edge detection menemukan piksel di mana intensitas gambar berubah tajam — batas antara objek dan latar, antara cahaya dan bayangan. Ini adalah salah satu operasi paling fundamental dalam CV klasik dan tetap relevan sebagai preprocessing untuk model modern.






Sobel operator


Menghitung gradien gambar di arah X dan Y secara terpisah. Magnitudo gradien: magnitude = sqrt(Gx² + Gy²). Mendeteksi tepi horizontal (Sobel Y) dan vertikal (Sobel X). Sensitif terhadap noise — biasanya didahului Gaussian blur.




Canny edge detector


Pipeline lengkap: (1) Gaussian blur, (2) Sobel gradient, (3) Non-maximum suppression (tipiskan tepi), (4) Double threshold (kuat/lemah), (5) Edge tracking by hysteresis. Hasilnya: tepi tipis dan akurat. cv2.Canny(img, low_thresh, high_thresh) adalah standar industri.








Pythonedge_detection.py


import cv2
import numpy as np

img = cv2.imread("foto.jpg", cv2.IMREAD_GRAYSCALE)

# Sobel — gradien X dan Y
sobel_x = cv2.Sobel(img, cv2.CV_64F, dx=1, dy=0, ksize=3)
sobel_y = cv2.Sobel(img, cv2.CV_64F, dx=0, dy=1, ksize=3)
magnitude = np.sqrt(sobel_x**2 + sobel_y**2)
magnitude = np.uint8(np.clip(magnitude, 0, 255))

# Canny — pipeline lengkap, direkomendasikan
blur = cv2.GaussianBlur(img, (5, 5), 0)
edges = cv2.Canny(blur, threshold1=50, threshold2=150)
# threshold1 = lower, threshold2 = upper untuk hysteresis

# Laplacian — deteksi tepi semua arah sekaligus
laplacian = cv2.Laplacian(img, cv2.CV_64F)
laplacian_abs = np.uint8(np.absolute(laplacian))

# Contour detection dari edges
contours, _ = cv2.findContours(edges, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
img_contours = cv2.drawContours(cv2.cvtColor(img, cv2.COLOR_GRAY2BGR),
                                  contours, -1, (0,255,0), 2)

1.4 Transformasi Geometris // resize, rotate, warp




Transformasi geometris mengubah posisi piksel dalam gambar. Ini penting untuk: preprocessing (resize ke ukuran model), augmentasi data (rotasi, flip), dan koreksi perspektif (dokumen scanner, lane detection mobil otonom).






Transformasi affine vs perspektif


Affine: mempertahankan paralelisme garis. Operasi: translate, rotate, scale, shear. Butuh 3 pasang titik korespondensi untuk estimasi matrix. Perspektif (homography): tidak mempertahankan paralelisme. Untuk koreksi "bird-eye view" atau rectifikasi dokumen miring. Butuh 4 pasang titik.




Interpolasi saat resize


INTER_NEAREST: cepat, blocky — untuk label mask segmentasi (jangan sampai nilai interpolasi memunculkan kelas baru). INTER_LINEAR: bilinear — default yang baik untuk upscaling. INTER_AREA: terbaik untuk downscaling. INTER_LANCZOS4: kualitas tertinggi, paling lambat.








Pythongeometric_transforms.py


import cv2
import numpy as np

img = cv2.imread("foto.jpg")
h, w = img.shape[:2]

# Resize ke ukuran tetap (untuk input model)
resized = cv2.resize(img, (224, 224), interpolation=cv2.INTER_LINEAR)

# Rotasi 45 derajat mengelilingi pusat
M_rot = cv2.getRotationMatrix2D(center=(w//2, h//2), angle=45, scale=1.0)
rotated = cv2.warpAffine(img, M_rot, (w, h))

# Horizontal flip (augmentasi data umum)
flipped = cv2.flip(img, flipCode=1)   # 1=horizontal, 0=vertical, -1=keduanya

# Koreksi perspektif — contoh dokumen miring
src_pts = np.float32([[50,50],[400,30],[430,380],[20,400]])
dst_pts = np.float32([[0,0],[400,0],[400,400],[0,400]])
M_persp = cv2.getPerspectiveTransform(src_pts, dst_pts)
warped = cv2.warpPerspective(img, M_persp, (400, 400))

# Padding gambar ke aspect ratio tertentu tanpa distorsi
def letterbox(img, target_size=640):
    h, w = img.shape[:2]
    scale = target_size / max(h, w)
    nh, nw = int(h*scale), int(w*scale)
    resized = cv2.resize(img, (nw, nh))
    top = (target_size - nh) // 2
    left = (target_size - nw) // 2
    return cv2.copyMakeBorder(resized, top, target_size-nh-top,
                               left, target_size-nw-left, cv2.BORDER_CONSTANT, value=114)











1.5 Color Spaces & Histograms // representasi warna yang tepat untuk setiap task




RGB bukan satu-satunya cara merepresentasikan warna. Color space yang tepat membuat banyak task menjadi jauh lebih mudah: segmentasi kulit lebih mudah di HSV, deteksi cahaya lebih mudah di LAB, dan kompresi gambar lebih efisien di YCbCr.






Color spaces penting


    

  • RGB/BGR — standar, natural untuk gambar
    • 

  • HSV (Hue, Saturation, Value) — H adalah "warna murni", mudah untuk filter warna (kuning, merah, dll)
    • 

  • LAB — perceptually uniform, L=lightness terpisah dari warna — bagus untuk color matching
    • 

  • YCbCr — Y=luminance, dipakai di JPEG/video compression
    • 

  • Grayscale — satu channel, hemat memori, cukup untuk banyak task texture/edge
    • 





Histogram gambar


Distribusi intensitas piksel per channel. Histogram yang ideal: tersebar merata (kontras baik). Histogram equalization: meregangkan distribusi untuk meningkatkan kontras — berguna untuk gambar gelap atau backlit. CLAHE (Contrast Limited AHE): local equalization yang lebih robust.








Pythoncolor_and_histogram.py


import cv2
import numpy as np

img_bgr = cv2.imread("foto.jpg")

# Konversi color space
img_hsv  = cv2.cvtColor(img_bgr, cv2.COLOR_BGR2HSV)
img_lab  = cv2.cvtColor(img_bgr, cv2.COLOR_BGR2Lab)
img_gray = cv2.cvtColor(img_bgr, cv2.COLOR_BGR2GRAY)

# Segmentasi warna kuning via HSV masking
lower_yellow = np.array([20, 100, 100])
upper_yellow = np.array([40, 255, 255])
mask = cv2.inRange(img_hsv, lower_yellow, upper_yellow)
result = cv2.bitwise_and(img_bgr, img_bgr, mask=mask)

# Histogram dan equalization
hist = cv2.calcHist([img_gray], [0], None, [256], [0,256])
equalized = cv2.equalizeHist(img_gray)

# CLAHE — lebih baik untuk gambar natural
clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8,8))
clahe_result = clahe.apply(img_gray)

# Mean & std normalisasi per channel (dipakai sebelum model)
mean = img_bgr.mean(axis=(0,1))   # [B_mean, G_mean, R_mean]
std  = img_bgr.std(axis=(0,1))
