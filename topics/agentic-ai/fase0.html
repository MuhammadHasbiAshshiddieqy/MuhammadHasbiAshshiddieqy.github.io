0.1 Agent Pertama dalam 5 Menit // langsung jalankan, pahami belakangan

Cara terbaik memahami agent adalah dengan menjalankannya dulu. Di sini kamu akan build agent minimal yang bisa memanggil tool — lengkap, bisa dijalankan, dan hanya butuh beberapa menit setup.

Kamu butuh: Python 3.10+, Anthropic API key (dari console.anthropic.com), dan koneksi internet.

Prerequisites — apa yang perlu kamu siapkan
  • Python 3.10+ — cek dengan python --version
  • Anthropic API key — daftar di console.anthropic.com, buat API key baru
  • Estimasi biaya — contoh di halaman ini menghabiskan sekitar $0.001–0.005 per run (sangat murah). Haiku adalah model termurah untuk percobaan
  • Pengetahuan Python — familiar dengan fungsi, dict, dan loop sudah cukup

Pythonfirst_agent.py
import anthropic
import json
import datetime

# ── Tools yang tersedia untuk agent ─────────────────────────────────────
def get_current_time() -> str:
    return datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")

def calculate(expression: str) -> str:
    try:
        result = eval(expression, {"__builtins__": {}})  # sandbox sederhana
        return str(result)
    except Exception as e:
        return f"Error: {e}"

# ── Tool schemas — format yang dipahami model ────────────────────────────
TOOLS = [
    {
        "name": "get_current_time",
        "description": "Mendapatkan waktu saat ini",
        "input_schema": {"type": "object", "properties": {}, "required": []}
    },
    {
        "name": "calculate",
        "description": "Menghitung ekspresi matematika. Contoh: '2 + 2', '100 * 0.1'",
        "input_schema": {
            "type": "object",
            "properties": {
                "expression": {"type": "string", "description": "ekspresi matematika"}
            },
            "required": ["expression"]
        }
    }
]

# ── Registry: nama tool → fungsi Python ─────────────────────────────────
TOOL_REGISTRY = {
    "get_current_time": get_current_time,
    "calculate": calculate,
}

# ── Agent loop ───────────────────────────────────────────────────────────
def run_agent(user_input: str) -> str:
    client = anthropic.Anthropic()  # baca ANTHROPIC_API_KEY otomatis
    messages = [{"role": "user", "content": user_input}]

    for _ in range(10):  # max 10 iterasi, hindari infinite loop
        response = client.messages.create(
            model="claude-haiku-4-5-20251001",  # model termurah
            max_tokens=1024,
            tools=TOOLS,
            messages=messages
        )

        if response.stop_reason == "end_turn":
            # Model selesai — ambil text terakhir
            for block in reversed(response.content):
                if hasattr(block, "text"):
                    return block.text
            return "(selesai tanpa teks)"

        if response.stop_reason == "tool_use":
            # Tambahkan response model ke history
            messages.append({"role": "assistant", "content": response.content})

            # Eksekusi setiap tool yang diminta
            tool_results = []
            for block in response.content:
                if block.type == "tool_use":
                    fn = TOOL_REGISTRY.get(block.name)
                    result = fn(**block.input) if fn else "Tool tidak ditemukan"
                    tool_results.append({
                        "type": "tool_result",
                        "tool_use_id": block.id,
                        "content": result
                    })

            # Inject hasil tool ke context untuk iterasi berikutnya
            messages.append({"role": "user", "content": tool_results})
            continue

        break  # stop_reason lain (max_tokens, dll)

    return "Agent berhenti tanpa jawaban final"

# ── Jalankan ─────────────────────────────────────────────────────────────
if __name__ == "__main__":
    jawaban = run_agent("Jam berapa sekarang? Dan berapa 15% dari 847?")
    print(jawaban)
Coba sendiri dulu sebelum lanjutSimpan file di atas, jalankan dengan python first_agent.py. Perhatikan output-nya — agent akan memanggil kedua tools dan merangkai jawaban. Coba ubah pertanyaan di baris terakhir dan lihat bagaimana agent bereaksi berbeda.

Apa yang terjadi di balik layar?

  • Iterasi 1: Model menerima pesan user → memutuskan perlu memanggil get_current_time dan calculate → stop_reason == "tool_use"
  • Harness: Eksekusi kedua fungsi Python, inject hasilnya ke messages sebagai tool_result
  • Iterasi 2: Model membaca hasil tools → merangkai jawaban akhir → stop_reason == "end_turn"
  • Itulah agent loop: model → tool call → result → model. Loop ini bisa jalan puluhan iterasi untuk task yang kompleks
