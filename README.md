## Cara Deploy (untuk reference)

Repo ini adalah **user GitHub Pages** (`username.github.io`), sehingga:

- Branch `main` → root `/` → live otomatis di `https://MuhammadHasbiAshshiddieqy.github.io`
- Tidak perlu setting tambahan di GitHub Pages
- Setiap push ke `main` akan auto-deploy dalam ~1 menit

